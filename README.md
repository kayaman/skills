# skills

A collection of Agent Skills for GitHub Copilot.

## Available Skills

| Skill | Description |
| ----- | ----------- |
| [aws-well-architected](skills/aws-well-architected/SKILL.md) | Enforces AWS Well-Architected Framework best practices across all six pillars. |
| [language-conventions-source-en-docs-br](skills/language-conventions-source-en-docs-br/SKILL.md) | Enforces language usage conventions for Brazilian teams (English code, Brazilian Portuguese docs). |
| [rfc2119](skills/rfc2119/SKILL.md) | Enforces correct usage of RFC 2119 requirement keywords (MUST, SHOULD, MAY, etc.). |
| [semver](skills/semver/SKILL.md) | Enforces Semantic Versioning 2.0.0 rules for version number management. |

## Usage

To activate a skill, reference its `SKILL.md` in your GitHub Copilot custom instructions or agent configuration.
