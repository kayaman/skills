## Topics to Cover

- The two GoF maxims: program to an interface, favor composition over inheritance
- All 23 GoF patterns: 5 Creational, 7 Structural, 11 Behavioral (intent, when to use, when NOT to use)
- Modern patterns beyond GoF: Dependency Injection, Null Object, Repository, Unit of Work, Specification
- Singleton criticism and modern alternatives (DI containers)
- Norvig's observation: 16 of 23 patterns simplified or eliminated by dynamic language features
- Pattern-itis and YAGNI: when NOT to use patterns
- Refactoring toward patterns (emergent use, not speculative)
